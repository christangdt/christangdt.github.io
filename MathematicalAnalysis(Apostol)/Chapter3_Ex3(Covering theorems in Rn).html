                      Covering theorems in Rn

                      Ex 3.17

                      If S⊆Rn, prove that the collection of isolated points of S is countable.

                      Proof

                      Let x be an isolated point of S, then we can find a ball B(x,r) such that B(x,r)∩S=x. For any two isolated points x1 and x2, we can modify two balls to make them not joint. Thus each isolated points of S corresponds to a ball, we can choose a rational q for each ball, it follows that the set of isolated points of S is equivalent to a subset of Q, thus countable.

                      Ex 3.18

                      Prove that the set of open disks in the xy-plane with center at (x,x) and radius x>0, x rational, is a countable covering of the set {(x,y):x>0,y>0}.

                      Proof

                      It is easy to see the set of open disks is countable. To prove it is a covering of {(x,y):x>0,y>0}, consider (x,y) with x>0,y>0, we further suppose x≤y, let q be a rational such that

                      y<q<2y−(y−x)2−1⟹2q−q<2y−(y−x)

                      then the point (q,q) lies above the point (y,y) in the xy-plane, and

                      ∥(q,q)−(x,y)∥≤∥(q,q)−(y,y)∥+∥(y,y)−(x,y)∥=2(q−y)+(y−x)<q

                      thus the point (x,y) lies in the open disk with center at (q,q) and radius q. The case when x>y can be similarly proved.

                      Ex 3.19

                      The collection F of open intervals of the form (1/n,2/n), where n=2,3,…, is an open covering of the open interval (0,1). Prove (without using Theorem 3.31) that no finite subcollection of F covers (0,1).

                      Proof

                      For any finite subcollection of F, we can find a largest N in all n which constitute such intervals (1/n,2/n), then the point 1/(N+1) is not covered by the finite subcollection.

                      Ex 3.20

                      Give an example of a set S which is closed but not bounded and exhibit a countable open covering F such that no finite subset of F covers S.

                      Solution

                      Let S={n:n∈Z}, then S is closed since S=R−∪k∈Z(k,k+1) is the complement of an open set. S is obviously not bounded. The covering F={(n−1/2,n+1/2):n∈Z} covers S, it is countable since F is equivalent to Z, but no finite subset of F covers S.

                      Ex 3.21

                      Given a set S in Rn with the property that for every x in S there is an n-ball B(x) such that B(x)∩S is countable. Prove that S is countable.

                      Proof

                      By the Lindelof covering theorem, for each x∈S we choose a ball B(x) such that B(x)∩S is countable, the collection F={B(x)∩S:x∈S} is a covering of S, and we can choose a countable subcollection F′ that covers S. It follows that S⊆⋃B∈F′B, since each B is countable and F′ is countable, we have ⋃B∈F′B being countable and so is S.

                      Ex 3.22

                      Prove that a collection of disjoint open sets in Rn is necessarily countable. Given an example of a collection of disjoint closed sets which is not countable.

                      Proof

                      For any open set in this disjoint open sets, we can find a point with all coordinates rational, thus the collection is equivalent to a subset of rational points in Rn, thus countable. The collection of sets {(a,0,…,0)} with 0≤a≤1 is a collection which is not countable, but each set consists only one point (a,0,…,0), thus is closed and disjoint.

                      Ex 3.23

                      Assume that S⊆Rn. A point x in Rn is said to be a condensation point of S if every n-ball B(x) has the property that B(x)∩S is not countable. Prove that if S is not countable, then there exists a point x in S such that x is a condensation point of S.

                      Proof

                      Assume each point in S is not a condensation point of S, then for each x∈S we can find at least one ball B(x) such that B(x)∩S is countable, then S is countable by Exercise 3.21, a contradiction.

                      Ex 3.24

                      Assume that S⊆Rn and assume that S is not countable. Let T denote the set of condensation points of S. Prove that: a) S−T is countable, b) S∩T is not countable, c) T is a closed set, d) T contains no isolated points.

                      Proof

                      a) S−T consists of points that are not condensation points of S, which satisfies the requirements of Exercise 3.21, thus is countable. b) Assume S∩T is countable, then S=(S−T)∪(S∩T) is countable, a contradiction. c) Let x∉T, then we can find a ball B(x) such that B(x)∩S is countable. Let the radius of B(x) to be r, then for any point y∈B(x,r/2), the ball B(y,r/2)⊂B(x,r), thus we have B(y,r/2)∩S is countable, which means y∉T, it follows that Rn−T is open. d) Assume p is an isolated point of T, then by definition, p∈T and p is not an accumulation point of T, thus we can find a ball B(p) which contains no point of T other than p. Since p∈T, we have B(p)∩S not countable, but for any point x∈B(p),x≠p, the point x is not a condensation point of S, which means ∃B(x) such that B(x)∩S is countable. The collection of all such B(x) covers B(p)−{p}, by the Lindelof covering theorem, there is a countable subcollection F that covers this set, thus B(p)−{p}⊂⋃B∈FB, or B(p)∩S⊂(⋃B∈FB∩S)∪{p}. Notice that F is countable and B∩S is countable for each B∈F, we conclude B(p)∩S is countable, this is a contradiction.

                      Ex 3.25

                      A set in Rn is called perfect if S=S′, that is, if S is a closed set which contains no isolated points. Prove that every uncountable closed set F in Rn can be expressed in the form F=A∪B, where A is perfect and B is countable. (Cantor-Bendixon theorem)

                      Proof

                      Let A be the set of condensation points of F, let B=F−A, then A is perfect and B is countable by Exercise 3.24.
