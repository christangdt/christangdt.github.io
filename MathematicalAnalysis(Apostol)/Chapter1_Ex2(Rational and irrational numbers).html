                          Rational and irrational numbers

                          Ex 1.7

                          Find the rational number whose decimal expansion is 0.3344444...

                          Solution

                          Write 0.3344444...=0.33333...+0.00111...., then we have

                          0.3344444...=13+1900=301900

                          Ex 1.8

                          Prove that the decimal expansion of x will end in zeros (or in nines) if, and only if, x is a rational number whose denominator is of the form 2n5m, where m and n are non-negative integers.

                          Solution

                          One direction is obvious, to prove the other direction, notice that the unique factorization of the decimal base 10 is 2⋅5, thus other factor in the denominator would yield a nonzero end, since if x ends in zero, then we shall find a integer n such that 10nx is an integer, but factors other than 2 and 5 will make it impossible.

                          Ex 1.9

                          Prove that 2+3 is irrational.

                          Solution

                          Assume 2+3=m/n, then 3=m/n−2 and thus

                          3=m2/n2−2m/n2+2⇒2=n2m(m2/n2−1)∈Q

                          but 2 is irrational, we can prove this by: assume 2=p/q,(p,q)=1, then p2=2q2, which means p is even, thus p=2k and 4k2=2q2, which shows 2k2=q2 and q is even, which contradicts (p,q)=1.

                          Ex 1.10

                          If a,b,c,d are rational and if x is irrational, prove that (ax+b)/(cx+d) is usually irrational. When do expcetions occur?

                          Solution

                          If (ax+b)/(cx+d) is rational, we can assume

                          ax+bcx+d=mn,n≠0

                          which means nax+nb=mcx+md, or (mc−na)x=nb−md, thus x is rational when mc≠na and nb≠md, this is a contradiction. Exceptions occur when mc=na,nb=md, that is when a=c=0,d≠0, or when b=d=0,c≠0, or when a/c=b/d,c≠0,d≠0.

                          Ex 1.11

                          Given any real x>0, prove that there is an irrational number between 0 and x.

                          Solution

                          If x is rational, then choose x/2, if x is irrational, choose x/2.

                          Ex 1.12

                          If a/b<c/d with b>0,d>0, prove that (a+c)/(b+d) lies between a/b and c/d.

                          Solution

                          It is easy to get

                          a+cb+d−ab=bc−ad(b+d)ba+cb+d−cd=ad−bc(b+d)b

                          the sign of bc−ad and ad−bc are different (except bc=ad), thus (a+c)/(b+d) lies between a/b and c/d. If bc=ad, then we have (a+c)/(b+d)=a/b=c/d, the conclusion still holds.

                          Ex 1.13

                          Let a and b be positive integers. Prove that 2 always lies between the two fractions a/b and (a+2b)/(a+b). Which fraction is closer to 2?

                          Solution

                          First we have

                          2−ab=2b−ab=(2b−a)(a+b)b(a+b)

                          and then

                          2−a+2ba+b=(a−2b)(2−1)a+b=(a−2b)(2−1)b(a+b)b

                          if 2≥a/b, then a≤2b, the second expression is no greater than zero, and vise versa, thus the lie between part is true. We further denote

                          d1=|2−ab|,d2=|2−a+2ba+b|

                          then if a<2b, there is

                          d1=(2b−a)(a+b)b(a+b),d2=(2b−a)(2−1)b(a+b)b

                          since a+b>b>(2−1)b, we have d1>d2, the second expression is closer. If a>2b, the second expression is closer, with a similar proof. If a=2b, we have d1=d2. In conclusion, the second expression is closer to 2.

                          Ex 1.14

                          Prove that n−1+n+1 is irrational for every integer n≥1.

                          Solution

                          Assume n−1+n+1∈Q, then since

                          n−1+n+1=2n+1−n−1∈Q

                          we have n+1−n−1∈Q, further we can have both n+1∈Q and n−1∈Q, but if n is a positive integer, then n+1 and n−1 cannot be perfect squares at the same time, use Theorem 1.10, at least one of n+1 and n−1 is irrational, a contradiction.

                          Ex 1.15

                          Given a real x and an integer N>1, prove that there exists integers h and k with 0<k≤N such that |kx−h|<1/N.

                          Solution

                          The N+1 numbers tx−[tx],t=0,1,…,N all lies in [0,1), thus at least two has distance less than 1/N, let us say that integers i,j with i>j makes

                          |ix−[ix]−(jx−[jx])|=|(i−j)x+[jx]−[ix]|<1/N

                          Let k=i−j and h=[ix]−[jx]. That 0<k≤N follows from i>j and i≤N.

                          Ex 1.16

                          If x is irrational prove that there are infinitely many rational numbers h/k with k>0 such that |x−h/k|<1/k2.

                          Solution

                          Use Exercises 1.15 we have, for an integer N>1, there exists integers h and k with 0<k≤N such that |kx−h|<1/N, which means

                          |x−hk|<1kN≤1k2

                          This is true for all N>1, thus we can find infinitely many h/k.

                          Ex 1.17

                          Let x be a positive rational number of the form

                          x=∑k=1nakk!,

                          where each ak is a nonnegative integer with ak≤k−1 for k≥2 and an>0. Let [x] denote the greatest integer in x. Prove that a1=[x], that ak=[k!x]−k[(k−1)!x] for k=2,…,n, and that n is the smallest integer such that n!x is an integer. Conversely, show that every positive rational number x can be expressed in this form in one and only one way.

                          Solution

                          Write

                          x=a1+∑k=2nakk!

                          to prove a1=[x], notice that

                          ∑k=2nakk!≤∑k=2nk−1k!=∑k=2n(1(k−1)!−1k!)=1−1n!<1

                          we get [x]=a1. To calculate ak, we have

                          k!x=k!∑j=1najj!=(∑j=1kk!j!aj)+(∑j=k+1nk!j!aj)

                          and

                          (k−1)!x=(k−1)!∑j=1najj!=(∑j=1k−1(k−1)!j!aj)+(∑j=kn(k−1)!j!aj)

                          Now since

                          ∑j=k+1nk!j!aj=k!∑j=k+1najj!≤k!∑j=k+1n(1(j−1)!+1j!)=k!(1k!−1n!)<1

                          we see that [k!x]=∑j=1kk!j!aj similarly [(k−1)!x]=∑j=1k−1(k−1)!j!aj thus

                          [k!x]−k[(k−1)!x]=∑j=1kk!j!aj−∑j=1k−1(k)!j!aj=k!k!ak=ak

                          To prove n is the smallest integer such that n!x is an integer, we see that for any integer k≤n:

                          k!x=k!∑j=1najj!=(∑j=1kk!j!aj)+(∑j=k+1nk!j!aj)

                          the first bracket is an integer, the second bracket is between 0 and 1, the second bracket vanishes only when k=n. Conversely, if given a positive x∈Q, choose n being the smallest integer such that n!x is an integer, we let a1=[x] and ak=[k!x]−k[(k−1)!x] for k=2,…,n, then a1,…,an is decided uniquely. We only need to show x=∑k=1nak/k!. This is clear since

                          ∑k=1nakk!=[x]+∑k=2n[k!x]−k[(k−1)!x]k!=[x]+∑k=2n([k!x]k!−[(k−1)!x](k−1)!)=[x]+[n!x]n!−[x]=[n!x]n!=n!xn!=x

                          and the proof is complete.
