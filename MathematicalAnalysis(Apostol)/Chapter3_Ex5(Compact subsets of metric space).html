              Compact subsets of a metric space

              Prove each of the following statements concerning an arbitrary metric space (M,d) and subsets S,T of M.

              Exercise 3.38

              Assume S⊆T⊆M. Then S is compact in (M,d) if and only if S is compact in the metric subspace (T,d).

              Proof

              Suppose S is compact in (M,d), let F be an open covering of S, say S⊆⋃A∈FA, where each A∈F is open in T. To find a finite covering of S in T, notice that by Theorem 3.33 we can find some A′ open in M for each set A∈F such that A=A′∩T. Call the collection of all sets A′ as F′, we have

              S⊆⋃A∈FA=⋃A′∈F′(A′∩T)⊆⋃A′∈F′A′

              We can then find a finite number of the sets A′ to cover S, use the fact that S⊆T we get

              S=(S∩T)⊆(A1′∪⋯∪Ap′)∩T=A1∪⋯∪Ap

              This shows that S is compact in (T,d).

              Conversely, suppose S is compact in (T,d), let F′ be an open covering of S, say S⊆⋃A′∈FA′, where each A′∈F is open in M. Then let A=A′∩T, use Theorem 3.33 we know A is open in T, and

              S=(S∩T)⊆(⋃A′∈FA′)∩T=⋃A′∈FA′∩T=⋃A′∈FA

              Thus we can find a finite number of the sets A cover S, as each A⊆A′, the corresponding finite number of sets A′ cover S. This completes the proof.

              Exercise 3.39

              If S is closed and T is compact, then S∩T is compact.

              Proof

              By Theorem 3.38, T is closed, thus S∩T is closed in (T,d), as T is compact, we see S∩T is compact in (T,d), use Exercise 3.38 we see S∩T is compact in (M,d).

              Exercise 3.40

              The intersection of an arbitrary collection of compact subsets of M is compact.

              Proof

              Let F be a collection of compact subsets of M, for each A∈F, A is closed by Theorem 3.38, thus ⋂A∈FA is closed in M, by Theorem 3.39, ⋂A∈FA is compact.

              Exercise 3.41

              The union of a finite number of compact subsets of M is compact.

              Proof

              Let F be a finite collection of compact subsets of M, for each A∈F, A is closed by Theorem 3.38, thus ⋃A∈FA is closed in M, by Theorem 3.39, ⋃A∈FA is compact.

              Exercise 3.42

              Consider the metric space Q of rational numbers with the Euclidean metric. Let S consist of all rational numbers in the open interval (a,b), where a and b are irrational. Then S is a closed and bounded subset of Q which is not compact.

              Proof

              S is obviously bounded. If q is an accumulation point of S, then we must have q∈(a,b), since assume q≤a, as a is irrational we get q<a, then the ball B(q,a−q) contains no point of S, a contradiction, and the same case when we assume q≥b. To see S is not compact, choose an integer N such that 1/N<b−a, let An=(a+1/n,b)∩Q which is an open interval in Q, then the collection ⋃n=N∞An covers (a,b)∩Q, thus covers S. Now assume a finite number of the sets An cover S, notice Am⊆An if m≤n, we can define M to be the maximum integer such that AM=(a+1/M,b) is in the finite subcover we assume, then S⊆AM. Choose q∈Q such that a<q<a+1/M, then q∈S but q∉AM, a contradiction.
