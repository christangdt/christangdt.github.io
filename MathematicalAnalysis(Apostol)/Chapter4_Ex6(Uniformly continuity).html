                    Uniform continuity

                    4.50

                    Prove that a function which is uniformly continuous on S is also continuous on S.

                    Proof

                    Let f be uniformly continuous on S to T, then ∀ε>0, there exist δ>0 such that dS(x,y)<δ means dT(f(x),f(y))<ε. Let x0∈S, then whenever dS(x,x0)<δ, we have dT(f(x),f(x0))<ε, this means f is continuous at x0.

                    4.51

                    If f(x)=x2 for x∈R, prove that f is not uniformly continuous on R.

                    Proof

                    Given ε0=1/2, and the pair of points (n,n+1), for any δ>0, if n is large enough, then n+1−n=1n+1+n<δ, but |f(n+1)−f(n)|=1>ε0.

                    4.52

                    Assume that f is uniformly continuous on a bounded set S in Rn. Prove that f must be bounded on S.

                    Proof

                    Let g be a function defined on S― with g(x)=f(x),x∈S and for x∈S′, define g(x)=limz→xf(z). The definition is valid, since if x∈S′, then limz→xf(z)=limn→∞f(xn) for any sequence xn∈S such that xn→x. For ε>0, from the uniformly continuity of f we have ∥f(a)−f(b)∥<ε whenever ∥a−b∥<δ for some δ, thus as long as m,n is large enough such that xm,xn∈B(x;δ/2), we have ∥f(xn)−f(xm)∥<ε, so the sequence {f(xn)} is a Cauchy sequence, thus must converge. By definition, g is continuous on S―, from S is bounded we can see S― is bounded, thus S― is compact in Rn. Then g(S―) is compact, thus is bounded, and g(S)⊆g(S―), this means g(S) is bounded.

                    4.53

                    Let f be a function defined on a set S in Rn and assume that f(S)⊆Rn. Let g be defined on f(S) with value in Rk, and let h denote the composite function defined by h(x)=g[f(x)] if x∈S. If f is uniformly continuous on S and if g is uniformly continuous on f(S), show that h is uniformly continuous on S.

                    Proof

                    For ∀ε>0, since g is uniformly continuous on f(S), we can find a δ1>0 such that ∥g(a)−g(b)∥<ε whenever a,b∈f(S),∥a−b∥<δ1, since f is uniformly continuous on S, we can find δ>0 such that ∥f(x)−f(y)∥<δ1 whenever x,y∈S,∥x−y∥<δ, this means

                    x,y∈S,∥x−y∥<δ⟹∥g(f(x))−g(f(y))∥<ε

                    So h is uniformly continuous on S.

                    4.54

                    Assume f:S→T is uniformly continuous on S, where S and T are metric spaces. If {xn} is any Cauchy sequence in S, prove that {f(xn)} is a Cauchy sequence in T.

                    Proof

                    That f is uniformly continuous on S means for ε>0, there exists δ>0 such that dS(x,y)<δ⟹dT(f(x),f(y))<ε. For this δ, we can find N such that if m,n>N, we can have dS(xm,xn)<δ, thus if m,n>N, we can have dT(f(x),f(y))<ε, thus {f(xn)} is a Cauchy sequence in T.

                    4.55

                    Let f:S→T be a function from a metric space S to another metric space T. Assume f is uniformly continuous on a subset A of S and that T is complete. Prove that there is a unique extension of f to A― which is uniformly continuous on A―.

                    Proof

                    Let g be the extension of f to A― by define g(x)=limy→xf(y) for x∈A′−A, with the proof of Exercise 4.52 we know that g(x)∈T. Given ε>0, we have dT(f(x),f(y))<ε whenever x,y∈S,dS(x,y)<δ, Now let a,b∈A―, to prove g is uniformly continuous we break into cases:

                    Case I: a,b∈A, then as long as dS(a,b)<δ we have dT(f(a),f(b))<ε.

                    Case II: one point in A and one point in A′−A, suppose a∈A and b∈A′−A, then first there exist δ′>0 such that dT(g(x),g(b))<ε if dS(x,b)<δ′, now let dS(a,b)<δ/2, then choose some x∈S which lies in B(b;δ/2) such that dS(x,b)<δ′, we can have

                    dS(a,x)≤dS(a,b)+dS(x,b)<δ/2+δ/2=δ

                    which implies dT(g(a),g(x))<ε, thus dT(g(a),g(b))<2ε.

                    Case III : a,b∈A′−A, then there exists δ1>0,δ2>0 such that

                    dS(x,a)<δ1⟹dT(g(x),g(a))<εdS(x,b)<δ2⟹dT(g(x),g(b))<ε

                    now let dS(a,b)<δ/3, choose some x1∈A which lies in B(a;δ/3) such that dS(x1,a)<δ1, choose some x2∈A which lies in B(b;δ/3) such that dS(x2,b)<δ2, notice that dS(x1,x2)<δ, then we have

                    dT(g(a),g(b))≤dT(g(a),g(x1))+dT(g(x1),g(x2))+dT(g(x2),g(b))≤ε+ε+ε=3ε

                    In conclusion, for any ε>0, we select the δ according to the uniformly continuity of f, then for g, we only need to limit dS(a,b)<δ/3 to guarantee dT(g(a),g(b))<3ε for any a,b∈A―, this shows g is uniformly continuous on A―.

                    To prove g is unique, any extension of f which is different from g must have different value on at least one accumulation point p of S. Then suppose h is an extension of f on A―, and h(p)≠g(p), let ε0=dT(h(p),g(p))/2, then there is a ball B(p;r) such that d(g(p),h(x))<ε0 whenever x∈B(p;r)∩A by the definition of g(p) and h(x)=f(x) on A. This means dT(h(x),h(p))>ε0 whenever x∈B(p;r)∩A, this means h is not continuous at p, thus not uniformly continuous on A―.

                    4.56

                    In a metric space (S,d), let A be a nonempty subset of S. Define a function fA:S→R by the equation

                    fA=inf{d(x,y):y∈A},x∈S

                    The number fA(x) is called the distance from x to A.

                    (a) Prove fA is uniformly continuous on S.

                    (b) Prove that A―={x:x∈S and fA(x)=0}.

                    Proof

                    (a) For ε>0, let δ=ε/2, if a,b∈S and d(a,b)<δ, then we first can find a′∈A such that d(a,a′)<fA(a)+ε/2, then

                    fA(b)≤d(b,a′)≤d(a,b)+d(a,a′)<fA(a)+ε

                    similarly we can have fA(a)<fA(b)+ε, thus |fA(a)−fA(b)|<ε whenever d(a,b)<δ, which means fA is uniformly continuous on S.

                    (b) If p∈A―, then fA(p)=0, otherwise in the ball B(p;fA(p)/2) there will be no point in A, a contradiction to p being an accumulation point of A. Conversely if fA(p)=0 for some p∈S, then for any r>0, there exist a∈A such that d(a,p)<r, thus p∈A―.

                    4.57

                    In a metric space (S,d), let A and B be disjoint closed subsets of S. Prove that there exist disjoint open subsets U and V of S such that A⊆U and B⊆V.

                    Proof

                    Let g(x)=fA(x)−fB(x), by Exercise 4.56, g is uniformly continuous on S, and we have

                    A={x∈S,fA(x)=0},B={x∈S,fB(x)=0}.

                    Also since A∩B=∅, if x∈A then x∉B, thus x∉B′, which means fB(x)>0, similarly fA(y)>0,y∈A, thus if we let U=g−1(−∞,0),V=g−1(0,+∞), then A⊆U and B⊆V, and both U and V are open by the continuity of g. To see U∩V=∅, assume s∈U∩V, then s∈U means g(s)<0, s∈V means g(s)>0, which is a contradiction.
