              Vector-valued functions

              5.30

              If a vector-valued function f is differentiable at c, prove that

              f′(c)=limh→01h[f(c+h)−f(c)].

              Conversely, if this limit exists, prove that f is differentiable at c.

              Proof

              We write f=(f1,…,fn), use addition and multiplication for vectors, we can have

              f′(c)=(f1′(c),…,fn′(c))=(limh→0f1(c+h)−f1(c)h,⋯,limh→0fn(c+h)−fn(c)h)=limh→01h(f1(c+h),…,fn(c+h))−limh→01h(f1(c),…,fn(c))=limh→01h[f(c+h)−f(c)]

              Conversely, if the limit exists and has value A, then A is a vector which can be written as A=(A1,…,An), this means for h sufficiently close to 0, we have

              |f(c+h)−f(c)h−A|<ε

              and for each k=1,…,n we have

              |fk(c+h)−fk(c)h−Ak|≤|f(c+h)−f(c)h−A|

              this shows that fk′(c) exists for each k, thus f is differentiable at c.

              5.31

              A vector-valued function f is differentiable at each point of (a,b) and has constant norm ∥f∥. Prove that f(t)⋅f′(t)=0 on (a,b).

              Proof

              Write f=(f1,…,fn), then f is differentiable on (a,b) means each fk is continuous and differentiable on (a,b). Thus for t∈(a,b), use differentiable at t we have

              f(t)⋅f′(t)=∑k=1nfk(t)fk′(t)=∑k=1n(fk(t)limx→tfk(x)−fk(t)x−t)=limx→t∑k=1n(fk(x)−fk(t)x−tfk(t))

              further use continuous at t we have

              f(t)⋅f′(t)=limx→t∑k=1n(fk(x)−fk(t)x−tfk(x))

              so we can have

              f(t)⋅f′(t)=12f(t)⋅f′(t)+12f(t)⋅f′(t)=12limx→t∑k=1n(fk(x)−fk(t)x−tfk(t))+12limx→t∑k=1n(fk(x)−fk(t)x−tfk(x))=12limx→t∑k=1n((fk(x)−fk(t))(fk(x)+fk(t))x−t)=12limx→t∑k=1n(fk2(x)−fk2(t)x−t)=12limx→t∑k=1n[fk2(x)−fk2(t)]x−t

              the function has constant norm ∥f∥ means ∥f(t)∥=∥f(x)∥, or ∑k=1n[fk2(x)−fk2(t)]=0, and the conclusion follows.

              5.32

              A vector-valued function f is never zero and has a derivative f′ which exists and is continuous on R. If there is a real function λ such that f′(t)=λ(t)f(t) for all t, prove that there is a positive real function u and a constant vector c such that f(t)=u(t)c for all t.

              Proof

              Let v(t)=∫0tλ(x)dx, then v′(t)=λ(t), write f=(f1,…,fn), then from f′(t)=λ(t)f(t) we get fk′(t)=λ(t)fk(t) for k=1,…,n, thus

              (e−v(t)fk(t))′=(e−v(t))′fk(t)+e−v(t)fk′(t)=e−v(t)(−λ(t)fk(t)+fk′(t))=0

              which means e−v(t)fk(t)=ck for some constant ck∈R. Let u(t)=ev(t)>0 and c=(c1,…,cn), we have

              u(t)c=ev(t)(e−v(t)f1(t),…,e−v(t)fn(t))=(f1(t),…,fn(t))=f(t)

              5.33

              Consider the function f defined on R2 by the following formulas:

              f(x,y)=xyx2+y2,if (x,y)≠(0,0)f(0,0)=0.

              Prove that the partial derivatives D1f(x,y) and D2f(x,y) exist for every (x,y) in R2 and evaluate these derivatives explicitly in terms of x and y. Also, show that f is not continuous at (0,0).

              Proof

              When (x,y)≠(0,0), we have

              D1f(x,y)=limt→xf(t,y)−f(x,y)t−x=limt→xtyt2+y2−xyt2+y2t−x=limt→xty(x2+y2)−xy(t2+y2)(t−x)(t2+y2)(x2+y2)=limt→xtxy(x−t)+y3(t−x)(t−x)(t2+y2)(x2+y2)=limt→xy3−txy(t2+y2)(x2+y2)=y3−x2y(x2+y2)2

              Similarly, we can have D2f(x,y)=x3−xy2(x2+y2)2. For the case (x,y)=(0,0), we have

              D1f(0,0)=limt→0f(t,0)−f(0,0)t−0=0,D2f(0,0)=0

              thus D1f(x,y) and D2f(x,y) exists in R2.

              To see f is not continuous at (0,0), notice that for any δ>0, let (x,y)=(δ2,δ2), then

              ∥(x,y)−(0,0)∥=δ/2,|f(x,y)−f(0,0)|=1/2
