                  Open and closed sets in R1 and R2

                  Ex 3.1

                  Prove that an open interval in R1 is an open set and that a closed interval is a closed set.

                  Proof

                  Let (a,b) be an open interval, we allow a=−∞ or b=+∞, for any point x∈(a,b), we set r=min(1,x−a,b−x), then the 1-ball B(x;r)⊆(a,b), thus x is an interior point of (a,b). Let [a,b] be a closed interval, then R−[a,b]=(−∞,a)∪(b,∞), which is open.

                  Ex 3.2

                  Determine all the accumulation points of the following sets in R1 and decide whether the sets are open or closed (or neither).

                  • a) All integers.

                  • b) The interval (a,b].

                  • c) All numbers of the form 1/n,n=1,2,3,…

                  • d) All rational numbers.

                  • e) All numbers of the form 2−n+5−m,m,n=1,2,3,…

                  • f) All numbers of the form (−1)n+(1/m),m,n=1,2,3,…

                  • g) All numbers of the form (1/n)+(1/m),m,n=1,2,3,…

                  • h) All numbers of the form (−1)n[1+(1/n)],n=1,2,3,…

                  Solution

                  a) No accumulation points, the set is closed. b) The accumulation points are all points in [a,b], the set is neither open nor closed. c) Only one accumulation point 0, the set is neither open nor closed. d) All real numbers are accumulation points, the set is neither open nor closed. e) Only one accumulation point 0, the set is neither open nor closed. f) Two accumulation points 1 and −1, the set is neither open nor closed. g) The accumulation points are the points 1/n,n=1,2,… and 0, the set is neither open nor closed. h) Two accumulation points 1 and −1, the set is neither open nor closed.

                  Ex 3.3

                  The same as Exercise 3.2 for the following sets in R2:

                  • a) All complex z such that |z|>1.

                  • b) All complex z such that |z|≥1.

                  • c) All complex numbers of the form (1/n)+(i/m),(m,n=1,2,…)

                  • d) All points (x,y) such that x2−y2<1.

                  • e) All points (x,y) such that x>0.

                  • f) All points (x,y) such that x≥0.

                  Solution

                  a) The accumulation points are all complex z such that |z|≥1. The set is open. b) The accumulation points are all complex z such that |z|≥1. The set is closed. c) One accumulation point (0,0). The set is neither open nor closed. d) The accumulation points are the points (x,y) such that x2−y2≤1. The set is open. e) The accumulation points are all points (x,y) such that x≥0. The set is open. f) The accumulation points are all points (x,y) such that x≥0. The set is closed.

                  Ex 3.4

                  Prove that every nonempty open set S in R1 contains both rational and irrational numbers.

                  Proof

                  S is nonempty means we can find x∈S, thus there exists a 1-ball B(x,r)⊆S, between the point x−r and x+r we can find rational number and irrational numbers.

                  Ex 3.5

                  Prove that the only sets in R1 which are both open and closed are the empty set and R1 itself. Is a similar statement true for R2?

                  Proof

                  The empty set is obviously both open and closed. And for R1, each point x is an interior point since (x−1,x+1)∈R1, and the adherent point of R1 is R1 which contains in R1. Suppose S is a nonempty set of R1 which is not equal to R1, and assume S is both open and closed, S is open means that S=⋃n=1∞In where each In is a component interval and In∩Im=∅ for n≠m. Since S is nonempty, at least one In is nonempty, write this In to be (a,b), then a∉S, otherwise a∈Im for some m, and as Im is open we can find (a−r,a+r)⊆Im, then a+r/2∈In∩Im. If a∉S, then a∈R1−S, since S is closed, R1−S is open, which means we can find (a−r′,a+r′)∉S, but for any r, we have In∩(a−r,a+r)≠∅, thus a contradiction.

                  A similar statement is true for R2.

                  Ex 3.6

                  Prove that every closed set in R1 is the intersection of a countable collection of open sets.

                  Proof

                  Let S be a closed set, then R1−S is open. If R1−S is nonempty, then by Theorem 3.11, we can write R1−S=⋃i=1∞Ii, all of Ii are component intervals, thus S=R1−⋃i=1∞Ii=⋂i=1∞(R1−Ii) Now consider each set R1−Ii, write Ii=(ai,bi), then R1−Ii=(−∞,ai]∪[bi,+∞), this set can be written as a countable intersection of open sets, i.e.

                  (−∞,ai]∪[bi,+∞)=⋂n=1∞((−∞,ai+1n)∪(bi−1n,+∞))

                  since the intersection of a countable intersections is still countable intersection, we finished the proof.

                  Ex 3.7

                  Prove that a nonempty, bounded closed set S in R1 is either a closed interval, or that S can be obtained from a closed interval by removing a countable disjoint collection of open intervals whose endpoints belong to S.

                  Proof

                  Since S is nonempty and bounded, we let a=minS and b=maxS, then a≤b. Clearly S⊆[a,b]. Consider the closed interval [a,b], if [a,b]⊆S, this means S is a closed interval, if not, as S is closed, we have R1−S open, we also have a−1∈R1−S and thus R1−S is nonempty, thus we can write R1−S as a union of a countable collection of disjoint component intervals of R1−S. We first prove that (−∞,a) and (b,∞) must lie in this collection as a component interval. Consider the component interval in this collection that starts with −∞, i.e. the interval (−∞,k), we must have k≥a because (−∞,a)⊆R1−S, but if k>a, it means a∈R1−S, or a∉S, a contradiction. Thus (−∞,a) must lie in this collection, the same with (b,∞). Delete (−∞,a) and (b,∞) from the collection, we see that all the other component intervals lies in [a,b], and subtract all these intervals gives S. It only remains to show that the endpoints of any of these intervals belongs to S. Let I=(c,d) be any component interval of R1−S which lies in [a,b], then a≤c<d≤b. Consider c first, if c=a we have c∈S, if c>a, assume c∉S, then c∈R1−S which is open, thus we can find r>0 such that (c−r,c+r)∈R1−S, this means (c−r,d)∈R1−S, a contradiction since (c,d) is a component interval, thus c must belong to S, similarly we can show d must belong to S.
