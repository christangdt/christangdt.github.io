                        Miscellaneous properties of the interior and the boundary

                        If A and B denote arbitrary subsets of a metric space M, prove that:

                        Exercise 3.43

                        int A=M−M−A―.

                        Proof

                        Let x∈int A, then there is a ball B(x;r)⊆A, which means the ball B(x;r)∩(M−A) is empty, so x is not an adherent point of M−A, this means int A⊆M−M−A―. Conversely, let y∉M−A―, then there is a ball B(y;r′) which contains no point of M−A, this means B(y;r′)⊆A and y is an interior point of A.

                        Exercise 3.44

                        int (M−A)=M−A―.

                        Proof

                        x∈int (M−A)⟺∃B(x;r)⊆M−A⟺∃B(x;r),B(x;r)∩A=∅⟺x∉A―⟺x∈M−A―

                        Exercise 3.45

                        int (int A)=int A.

                        Proof

                        Let S=int A, then int S⊆S. We only need to prove S⊆int S. Let x∈S, then exists a ball B(x;r)⊆A, let r′=r/2, consider the ball B(x;r′), for each point y of this ball, we see the smaller ball B(y,r/4) is contained in B(x;r), since

                        d(z,x)≤d(z,y)+d(y,x)≤r2+r4<r,∀q∈B(y;r/4)

                        this shows that each y in B(x;r′) is an interior point of A, or y∈S, thus x∈int S, and the proof is complete.

                        Exercise 3.46

                        a) int (⋂i=1nAi)=⋂i=1n(int Ai), where each Ai⊆M.

                        b) int (⋂A∈FA)⊆⋂A∈F(int A), if F is an infinite collection of subsets of M.

                        c) Give an example where equality does not hold in (b).

                        Proof

                        a) First let x∈int (⋂i=1nAi), then exists a ball B(x;r)⊆⋂i=1nAi, thus B(x;r)⊆Ai for each i=1,…,n, which means x∈int Ai for each i=1,…,n, which means x∈⋂i=1n(int Ai), thus int (⋂i=1nAi)⊆⋂i=1n(int Ai). Conversely, let x∈⋂i=1n(int Ai), then x∈int Ai for i=1,…,n, thus there exist balls B(x;r1),…,B(x;rn) such that B(x;ri)⊆Ai for each i. Choose r′=min(r1,…,rn), we see that B(x;r′)⊆Ai for each i, thus B(x;r′)⊆⋂i=1nAi, which means x∈int (⋂i=1nAi), thus int (⋂i=1nAi)⊇⋂i=1n(int Ai).

                        Exercise 3.47

                        a) ⋃A∈F(int A)⊆int (⋃A∈FA).

                        Exercise 3.48

                        a) int (∂A)=∅ if A is open or if A is closed in M.

                        b) Give an example in which int (∂A)=M.

                        Proof

                        a) Assume there exists x∈M such that x∈int (∂A), then there is a ball B(x;r) such that B(x;r)⊆∂A. From this we get x∈∂A, by definition, B(x;r) contains at least one point p∈A and one point q∈M−A. Now if A is open, p shall be an interior point of A, we can find a ball B(p;r′) which contained in B(x;r) such that B(p;r′)⊆A. But from B(x;r)⊆∂A we know that p∈∂A, thus B(p;r′) shall contain a point of M−A, this is a contradiction. If A is closed, then M−A is open, with the same logic applied to q we get another contradiction.

                        Exercise 3.49

                        If int A=int B=∅ and if A is closed in M, then int (A∪B)=∅.

                        Proof

                        If either A or B is empty the proof is trivial. Suppose both A and B are not empty, and assume there is x∈int (A∪B), then we can find a ball B(x;r)⊆A∪B, if A is closed, then M−A is open, thus S=B(x;r)∩(M−A) is an open set and S⊆B. If S is empty, then B(x;r)⊆A and x∈int A, a contradiction to int A=∅. If S is a nonempty open set in B, choose p∈S, there is a ball B(p;r′)⊆S⊆B, then p∈int B, a contradiction to int B=∅. We see both case lead to a contradiction, and the proof is complete.

                        Exercise 3.51

                        ∂A=A―∩M−A― and ∂A=∂(M−A).

                        Proof

                        By definition, x∈∂A means that every ball B(x;r) contains at least one point of A and at least one point of M−A, this means x∈A― and x∈M−A―. Conversely, if x∈A―∩M−A―, then every ball B(x;r) contains at least one point of A and at least one point of M−A, and then x∈∂A.

                        To prove the second statement, notice that

                        ∂(M−A)=M−A―∩M−(M−A)―=M−A―∩A―=∂A

                        Exercise 3.52

                        If A―∩B―=∅, then ∂(A∪B)=∂A∪∂B.

                        Proof

                        By Exercise 3.12 we have (A∪B)′=A′∪B′ and A― is the smallest closed set containing A. We first show that A⊆M−B―, for any a∈A, assume a∉M−B―, then there exists a ball B(a;r) which has no points of M−B, then B(a;r)⊆B, in particular, a∈B, this means a∈A∩B⊆A―∩B―, a contradiction to A―∩B―=∅. Thus A⊆M−B―, and we further get A―⊆M−B―, as M−B― is closed. By symmetry we can get B―⊆M−A―.

                        Next, we show M−(A∪B)―=M−A―∩M−B―. Suppose x∈M−(A∪B)―, then every ball B(x;r) contains at least one point of M−(A∪B). Assume x∉M−A―, then there exists some ball B(x;r0) which has no point of M−A, or B(x;r0)⊆A⊆A∪B, a contradiction, thus we must have x∈M−A―, similarly we could get x∈M−B― and so M−(A∪B)―⊆M−A―∩M−B―. Conversely, let x∈M−A―∩M−B―, and assume x∉M−(A∪B)―, then there exists a ball B(x;r1)⊆A∪B, we claim that for some r′<r1, either B(x;r′)⊆A or B(x;r′)⊆B, if not, then any ball B(x;r) with r<r1 will contains at least one point of A and at least one point of B, this means x∈A―∩B―, which contradicts the condition A―∩B―=∅. But B(x;r′)⊆A means x∉M−A― and B(x;r′)⊆B means x∉M−B―, both leads to a contradiction. So we get M−A―∩M−B―⊆M−(A∪B)―.
