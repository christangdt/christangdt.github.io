                      4.1

                      Prove each of the following statements about sequences in C.

                      a) zn→0 if |z|<1; {zn} diverges if |z|>1.

                      b) If zn→0 and if {cn} is bounded, then cnzn→0.

                      c) zn/n!→0 for every complex z.

                      d) If an=n2+2−n, then an→0.

                      Solution

                      We let z=a+bi for a,b∈R in (a),(b) and (c).

                      a) If |z|<1, then |zn−0|=|zn|=|z|n→0. Thus zn→0. If |z|>1, then |zn|=|z|n→∞, thus zn diverges.

                      b) We can find M∈R+ such that |cn|≤M, then |cnzn|≤M|zn|→0.

                      c) Since |zn/n!|≤|z|n/n! and an/n!→0 for any real number a≥0.

                      d) Since 0<an=2n2+2+n<2n→0.

                      If 0<x1<1 and if xn+1=1−1−xn for all n≥1, prove that {xn} is a decreasing sequence with limit 0. Prove also that xn+1/xn→12.

                      Solution

                      xn+1xn=1−1−xnxn=1−(1−xn)xn(1+1−xn)=11+1−xn

                      4.5

                      A real sequence xn satisfies 7xn+1=xn3+6 for n≥1, if x1=1/2, prove that the sequence increases and find its limit. What happens if x1=3/2 or if x1=5/2?

                      Solution

                      If xn<1, then xn3<1, which gives xn+1<1, thus we can conclude xn<1 for all n if x1=1/2, and from 7xn+1=xn3+6 we get

                      xn+1−xn=(xn−1)(xn2−6)7>0

                      thus the sequence is increasing, since it is bounded, it must converge, let xn→a, we see that 7a=a3+6, thus a=1 since 0<xn<1 limits the possible value of a.

                      If x1=3/2, then it is easy to see that xn+1<xn and 1<xn<3/2 for all n≥2, thus the sequence decreases, and its limit is also 1.

                      If x1=5/2, then xn increases, and the sequence has no limit.

                      If |an|<2 and |an+2−an+1|≤18|an+12−an2| for all n≥1, prove that {an} converges.

                      4.7

                      In a metric space (S,d), assume that xn→x and yn→y. Prove that d(xn,yn)→d(x,y).

                      Solution

                      For ∀ε>0, we can find N>0 such that d(xn,x)<ε/2,d(yn,y)<ε/2 if n>N, we also have

                      d(xn,yn)≤d(xn,x)+d(x,y)+d(y,yn)<d(x,y)+εd(x,y)≤d(x,xn)+d(xn,yn)+d(yn,y)<d(xn,yn)+ε

                      which shows |d(x,y)−d(xn,yn)|<ε when n>N.

                      Prove that in a compact metric space (S,d), every sequence in S has a subsequence which converges in S. This property also implies that S is compact but you are not required to prove this.

                      nk=min{n>nk−1:d(xnk,p)<1k}

                      4.9

                      Let A be a subset of a metric space S. If A is complete, prove that A is closed. Prove that the converse also holds if S is complete.

                      Solution

                      Let p be an accumulation point of A, then there is a sequence {an} in A which converges to p by choosing an∈A inside the ball B(p;1/n), since A is complete, we see that p∈A, and thus A contains all its accumulation points, so A is closed.

                      If A is closed and S is complete, let {an} be a Cauchy sequence in A, then it converges in S to some point p, by definition, p is an accumulation point of A, thus p∈A, this means A is complete.
