          Limits of functions

          NOTE. In Exercises 4.10 through 4.28, all functions are real-valued.

          4.10

          Let f be defined on an open interval (a,b) and assume x∈(a,b). Consider the two statements:

          (a)limh→0|f(x+h)−f(x)|=0;(b)limh→0|f(x+h)−f(x−h)|=0

          Prove that (a) always implies (b), and give an example in which (b) holds but (a) does not.

          Proof

          For ∀ε>0, there exists δ>0 such that |f(x+h)−f(x)|<ε/2 for |h|<δ, thus

          |f(x+h)−f(x−h)|≤|f(x+h)−f(x)|+|f(x−h)−f(x)|<ε

          for every h which satisfies |h|<δ.

          Let f(x)=0 if x≠0 and f(0)=1, then limh→0|f(h)−f(−h)|=0 but

          limh→0|f(0+h)−f(0)|=1

          4.11

          Let f be defined on R2. If

          lim(x,y)→(a,b)f(x,y)=L

          and if the one-dimensional limits limx→af(x,y) and limy→bf(x,y) both exists, prove that

          limx→a[limy→bf(x,y)]=limy→b[limx→af(x,y)]=L

          Proof

          Let limx→af(x,y)=A(y) and limy→bf(x,y)=B(x), for ∀ε>0, first there is a δ1>0 such that

          |f(x,y)−L|<ε2whenever |(x,y)−(a,b)|<δ1

          and there is a 0<δ2<δ1 such that

          |f(x,y)−A(y)|<ε2,∀ywhenever |x−a|<δ2

          let δ=δ1−δ2, then if |y−b|<δ, we can have

          |(x,y)−(a,b)|≤|x−a|+|y−b|<δ2+δ=δ1

          thus as long as x is close enough to a, we have

          |A(y)−L|≤|f(x,y)−A(y)|+|f(x,y)−L|<ε,whenever |y−b|<δ

          this means limy→bA(y)=L, similarly we can prove limx→aB(x)=L.

          4.12

          If x∈[0,1] prove the following limit exists,

          limm→∞[limn→∞cos2n⁡(m!πx)],

          and that its value is 0 or 1, according to whether x is rational or irrational.

          Proof

          If x is irrational, then m!x cannot be an integer, thus |cos⁡(m!πx)|<1 for any possible choices of m, thus limn→∞cos2n⁡(m!πx)=0 and the limit we seek is 0.

          If x is rational, say x=p/q with p,q>0 and p,q have no common factors, then

          |cos⁡(m!πx)|{=1,m≥q<1,m<q

          thus when m≥q, the limit has value 1, when m<q, the limit has value 0.
