                  Ex 2.10

                  Prove that if A∼B and B∼C, then A∼C.

                  Proof

                  Since A∼B and B∼C, we can find one-to-one functions F and G, with F(A)=B and G(B)=C, thus G∘F is a one-to one function from A onto C.

                  Ex 2.11

                  If {1,2,…,n}∼{1,2,…,m}, prove that m=n.

                  Proof

                  Assume m<n, then let F be the one-to-one function with F{1,2,…,n}={1,2,…,m}, since there are n possible variables to choose and the value can have only m possible outcomes, F cannot be one-to-one, this is a contradiction. The case m>n can be similarly proved.

                  Ex 2.12

                  If S is an infinite set, prove that S contains a countably infinite subset.

                  Proof

                  That S is infinite means S is not emtpy, choose a∈S, consider the set S∖{a}, assume this set is finite, then we have some integer n such that S∖{a}∼{1,2,…,n}, let F be the one-to-one function with F(S∖{a})={1,2,…,n}. Define G with G=F on S∖{a} and G(a)=n+1, we can see that G is a one-to-one function with G(S)={1,2,…,n+1}, this means S is finite, a contradiction. Thus S∖{a} must be infinite. We let this a=a1, and by the same argument, we can choose a2 from S∖{a1}, repeat the step we can choose a3,a4,… and the set {a1,a2,…} is a countably infinite subset of S.

                  Ex 2.13

                  Prove that every infinite set S contains a proper subset similar to S.

                  Proof

                  S is infinite means S is either countably infinite or uncountable. If S is countably infinite, then S is similar to {1,2,…,}, let F be the one-to-one function with F({1,2,…,})=S, let G be the such that G(n)=F(n+1) for natural numbers n, then F∘G is one-to-one from {1,2,…,} to S∖{F(1)}, which means F∘G∘F−1 is one-to-one from S to S∖{F(1)}, thus S is similar to S∖{F(1)}. If S is uncountable, then S contains a countably infinite subset by Exercise 2.12, let this subset be A, choose a∈A, use the technique above we can prove that A is similar to A∖{a}. Let F be the one-to-one function such that F(A)=A∖{a}, let G be the function such that G=F on A and G(s)=s if s∈S−A, then G is one-to-one with G(S)=S∖{a}, thus S∖{a} is the proper subset we need.

                  Ex 2.14

                  If A is a countable set and B an uncountable set, prove that B−A is similar to B.

                  Proof

                  The set B∩A is countable. Thus the set B−A must be infinite, otherwise B is countable. By Exercise 2.12, we can choose a countably infinite subset C of B−A and the set S=C∪(B∩A) is still countably infinite. Notice that B=(B−A)∪(B∩A), we can define a one-to-one function F as follows: Since C and C∪(B∩A) are both countable, they are similar, thus we find a one-to-one function F′ such that F′(C)=C∪(B∩A), we let F=F′ on C, and F(a)=a for all a∈(B−A)−C, then F is a one-to-one function from B−A to B.

                  Ex 2.15

                  A real number is called algebraic if it is a root of an algebraic equation f(x)=0, where f(x)=a0+a1x+⋯+anxn is a polynomial with integer coefficients. Prove that the set of all polynomials with integer coefficients is countable and deduce that the set of algebraic numbers is also countable.

                  Proof

                  Let An be the set of all polynomials with integer coefficients and with degree n, i.e.

                  An={a0+a1x+⋯+anxn:n∈N}

                  Since there are countable choicesfor each ai with 0≤i≤n, the set An is countable as it is a finite collection of countable sets, thus ∪n=1∞An is also countable. For each f∈An, the total amount of algebraic numbers x which makes f(x)=0 is no more than n, which means the set of algebraic numbers that belongs to polynomial with integer coefficients with degree n is countable, and the conclusion follows.

                  Ex 2.16

                  Let S be a finite set consisting of n elements and let T be the collection of all subsets of S. Show that T is a finite set and find the number of elements in T.

                  Solution

                  There are 2n elements in T. Since for each of the n elements in S, it may or may not be in any subset of S.
