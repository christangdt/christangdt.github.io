                          Define T:P(R)→RR by

                          Tp={p−p(3)x−3,if x≠3,p′(3)if x=3.

                          Show that Tp∈P(R) for every polynomial p∈P(R) and that T is a linear map.

                          Solution: Let q(x)=p(x)−p(3), which is a polynomial, since q(3)=p(3)−p(3)=0, we see that 3 is a zero of q, thus q(x)=(x−3)r(x) for some r∈P(R). Thus Tp=r(x) when x≠3, which is a polynomial. To show that Tp(3)=r(3), notice that when x≠3 we have

                          limx→3r(x)=limx→3p(x)−p(3)x−3=p′(3)

                          so Tp is continuous at x=3, thus Tp is a polynomial.

                          To show T is a linear map, for p,q∈P(R) and x≠3 we have

                          T(p+q)(x)=p(x)+q(x)−p(3)−q(3)x−3=p(x)−p(3)x−3+q(x)−q(3)x−3=Tp(x)+Tq(x)T(cp)(x)=cp(x)−cp(3)x−3=cp(x)−p(3)x−3=cTp(x)

                          T(p+q)(3)=(p+q)′(3)=p′(3)+q′(3)=Tp(3)+Tq(3)T(cp)(3)=(cp)′(3)=cp′(3)=cTp(3)

                          so we conclude T(p+q)=Tp+Tq and T(cp)=cTp on P(R), so T is linear.

                          11

                          Suppose p∈P(F) with p≠0. Let U={pq:q∈P(F)}.

                          (a) Show that dim⁡P(F)/U=deg⁡p.

                          (b) Find a basis of P(F)/U.

                          Solution:

                          We have P(F)/U={r+U:r∈P(F)}, let deg⁡p=m and let Ui=zi+U for i=0,…,m−1, we show that U0,…,Um−1 is a basis of P(F)/U, this can prove both (a) and (b).

                          It is easy to see that U0,…,Um−1 is linearly independent in P(F)/U. To see it spans P(F)/U, we let r+U be any element in P(F)/U, use Division Algorithm for Polynomials, we can find s,t∈P(F) such that r=sp+t and deg⁡t<deg⁡p, so we have

                          r+pq=sp+t+pq=t+(s+q)p

                          this means r+U=t+U, since deg⁡t<deg⁡p, t can be represented by 1,z,…,zm−1, so r+U is a linear combination of U0,…,Um−1, and the proof is done.
